Windows Server 重大漏洞修補通知

Microsoft 於 10月份安全更新中修補了多個高危漏洞，其中包括可能導致遠端程式碼執行的漏洞。 建議所有使用 Windows Server 的醫療機構立即進行更新。

針對醫療產業的勒索軟體攻擊浪潮

📅 2025-10-24 🔍 來源：國際資安組織
高風險

近期觀察到多起針對醫療機構的勒索軟體攻擊事件，攻擊者利用釣魚郵件和已知漏洞入侵醫院網路， 加密重要醫療資料並要求贖金。受害機構包括美國、歐洲及亞洲地區的多家醫院。

攻擊手法：

透過魚叉式釣魚郵件傳送惡意附件，或利用未修補的 VPN 漏洞取得初始存取權限， 再使用橫向移動技術擴散至整個網路環境。

防護建議：
  • 立即更新所有系統安全修補程式
  • 強化郵件安全閘道，加強釣魚郵件過濾
  • 實施網路隔離，限制橫向移動
  • 定期進行離線備份，並測試還原程序
  • 加強員工資安意識訓練
#勒索軟體 #醫療產業 #釣魚攻擊

醫療 IoT 設備存在未修補漏洞

📅 2025-10-22 🔍 來源：IoT 安全研究機構
中風險

研究人員發現多款醫療 IoT 設備存在安全漏洞，包括輸液幫浦、生理監視器等設備。 這些漏洞可能被用來竄改設備設定或竊取病患資料。部分設備製造商已釋出更新， 但許多醫院尚未完成修補。

受影響設備：

特定品牌的輸液幫浦、生理監視器、呼吸器控制系統。詳細清單請參考製造商公告。

防護建議：
  • 盤點所有醫療 IoT 設備，確認韌體版本
  • 將醫療設備隔離於獨立 VLAN
  • 限制醫療設備的網路存取權限
  • 與設備供應商聯繫，安排修補更新
  • 加強設備存取日誌監控
#IoT安全 #醫療設備 #漏洞管理

新型 SQL Injection 攻擊技術

📅 2025-10-20 🔍 來源：Web 安全研究團隊
高風險

安全研究人員發現新型的 SQL Injection 攻擊技術，可繞過常見的 WAF 防護。 此技術已被用於攻擊多個醫療機構的線上掛號系統和病患查詢系統， 造成敏感資料外洩風險。

攻擊特徵：

利用編碼變換和時間盲注技術繞過輸入驗證，針對舊版 Web 應用程式的資料庫查詢介面進行攻擊。

防護建議：
  • 檢視所有對外 Web 應用程式的輸入驗證機制
  • 採用參數化查詢（Prepared Statements）
  • 更新 WAF 規則，加強 SQL Injection 防護
  • 實施最小權限原則，限制資料庫帳號權限
  • 定期進行 Web 應用程式漏洞掃描
#SQL Injection #Web安全 #資料外洩

社交工程攻擊瞄準醫療人員

📅 2025-10-18 🔍 來源：資安事件通報
中風險

近期發現多起針對醫療人員的社交工程攻擊，攻擊者冒充 IT 部門人員或醫療器材廠商， 透過電話或電子郵件要求提供帳號密碼或安裝遠端存取軟體。 已有數家醫院的員工受騙，導致帳號被盜用。

常見手法：

冒充 IT 支援人員聲稱需要進行系統維護，要求提供帳號密碼； 或冒充醫療器材廠商要求安裝更新程式，實際為遠端控制木馬。

防護建議：
  • 加強員工資安意識訓練，特別是社交工程防範
  • 建立明確的身份驗證程序，IT 人員不會索取密碼
  • 實施多因素認證（MFA）
  • 設立資安事件通報管道
  • 定期進行社交工程演練測試
#社交工程 #釣魚攻擊 #帳號盜用

過時的 SSL/TLS 協定安全風險

📅 2025-10-15 🔍 來源：加密技術研究
低風險

安全掃描發現部分醫療機構仍在使用已過時的 SSL/TLS 協定版本（如 TLS 1.0、1.1）， 這些協定存在已知的安全弱點，可能被用於中間人攻擊。 雖然風險相對較低，但建議盡快升級至 TLS 1.2 或 1.3。

潛在風險：

過時的協定可能被攻擊者利用進行降級攻擊，截取或竄改加密通訊內容。

防護建議：
  • 掃描並盤點所有使用 SSL/TLS 的服務
  • 停用 TLS 1.0 和 1.1，僅支援 TLS 1.2 及以上版本
  • 更新伺服器和應用程式以支援新版協定
  • 檢查並更新 SSL 憑證設定
  • 使用強加密套件
#SSL/TLS #加密協定 #系統升級
