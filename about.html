📋 個人簡介

我是羅，目前在醫療機構擔任資訊工程師，主要負責網路架構與資訊安全的維運工作。 每天的工作包含網路設備管理、資安事件處理、系統監控與故障排除。 在醫療環境中，資訊系統的穩定與安全直接關係到病患照護品質，這讓我深刻體會到資訊人員的責任重大。 透過這個部落格，我想分享在第一線工作中遇到的實際問題與解決經驗， 希望能與同樣在醫療資訊領域奮鬥的夥伴們互相交流學習。

🎯 專業領域

網路架構與安全

  • 醫院網路架構規劃與維運（多層 VLAN、網路隔離）
  • Cisco 網路設備管理與設定
  • 防火牆規則配置與管理
  • F5 負載平衡器操作與維護
  • VPN 與遠端存取安全管理

資訊安全實務

  • 資安事件監控與緊急應變
  • 弱點掃描與漏洞修補追蹤
  • 滲透測試與紅藍隊演練協助
  • 日誌分析與異常行為偵測
  • 資安政策執行與稽核配合
  • 員工資安教育訓練

系統管理

  • Windows Server / Linux 伺服器管理
  • Active Directory 維運與 LDAPS 整合
  • SSL/TLS 憑證申請與配置

🎓 持續成長

資訊技術日新月異，資安威脅也不斷演進。我持續透過線上課程、技術文件、 社群交流來學習新知識。也會參加資安研討會、CTF 競賽來精進實戰技能。 這個部落格不只是分享，也是我整理思緒、記錄成長的地方。 很多時候，把問題寫下來的過程，就能找到更好的解決方案。
